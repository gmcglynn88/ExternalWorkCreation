How to Use This Tool

1. Choose a Queue
Use the dropdown to select the queue the proxy interaction should be created in.

2. Choose a User
Pick the agent who will be assigned the created interaction.

3. Add an External Reference (Optional)
Enter anything useful such as a case ID or URL. Leave blank if not needed.

4. Create the Proxy Interaction
Click Create Proxy Interaction. The tool will create an inbound email, assign it to the selected user, and disconnect it.

5. Review the Result
A success or error message will appear depending on the outcome.

6. Repeat as Needed
You can generate as many proxy interactions as you need.
